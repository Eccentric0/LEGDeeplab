
# Security Policy for LEGDeeplab

## Reporting a Vulnerability
If you discover a security vulnerability in LEGDeeplab, please report it privately to our security team at:

- **Email**: security@legdeeplab.org
- **Response Time**: We will acknowledge your report within 48 hours.

## Security Best Practices
- **Input Validation**: Always sanitize inputs from external sources.
- **Dependency Security**: Regularly update dependencies and perform vulnerability scanning.

## Security Testing
We use automated security testing tools to ensure code safety, including:
- Static analysis for code vulnerabilities
- Dependency scanning for known security flaws
